Cara mengompile program:

Windows:
g++ main.cpp oop.cpp -std=c++11 -Wall -O2 -mwindows -lmingw32 -lSDL2main -lSDL2 -lSDL2_image -lSDL2_ttf -w -o driv

Linux:
g++ main.cpp oop.cpp -std=c++11 -Wall -O2 -lSDL2main -lSDL2 -lSDL2_image -lSDL2_ttf -w -o driv

Cara memainkan game:

Pada menu, pilih tombol "play" dengan menggunakan tombol atas dan bawah, kemudian ketik tombol spasi, maka Anda akan memulai permainan

Anda menang jika berhasil mengumpulkan 3 egg

Anda kalah jika Anda tidak memiliki ikan dan uang Abda tidak cukup untuk membeli ikan baru
